Web 102

Class 1

In this class you will learn about the technologies that are used to create a functioning website. You will also learn about how information is transferred through the internet and the tools that can be used to debug and find problems in a web page.

Topics Covered

  • Introduction to the Web Stack
  • Introduction to HTTP
  • Using Firebug to get view requests and responses
  • Trace Route

Students should...

  • Know the technologies that are used to create a website
  • Have a brief understanding of how information is transferred through the internet
  • Understand tools that can be used for debugging and identifying problems

Class 2

In this class we will continue to discuss how information is transferred through the internet. We will also begin to discuss collaboration and how working with a team is beneficial when working on a web development project.

Topics Covered

  • GET and POST Requests
  • Discuss the class Project
  • Form Teams
  • Explore git branches used for the project

Students should...

  • Further understand how information is transferred through the internet
  • Understand the process of collaboration and why teams are important for development.

Class 3

In this class you will learn how data is passed within web applications between different technologies. You will also learn how third party libraries can help improve the development process.

Topics Covered

  • HTML Forms - Three essential elements
  • Review GET vs POST
  • jQuery and BootStrap Integration

Students should...

  • Understand how data is passed within web applications between different technologies.
  • Understand how third party libraries can help improve the development process.

Class 4

In this class you will review everything you learned in the course through combining each team's elements to create a functioning, deployed web application. You will also learn about cookies, sessions, and ajax. You will also have the opportunity to ask the instructor any remaining questions.

Topics Covered

  • Code Review
  • Branch Merging
  • Web Application Deployment
  • Cookies, Sessions, and Ajax

Students should...

  • Be able to see how everything they have learned fits together.
  • Understand how websites access user information.

FAQ

Why should I take Web Basics 102?

This intermediate-level course is essential to advance your understanding of professional web development. This course will dive you deeper into Web development, allowing students to work towards increasing their skill set. Anyone interested in a continuing to expand their understanding of Web development should take this course.

What will I learn?

Throughout this intermediate course you will learn more advanced design and website concepts using CSS, jQuery, PHP and MySQL to build more dynamic web pages.

What can I expect?

Each 2.5 hour session is a mixture of Lecture and hands-on Exercises to best teach you the fundamentals of Web design. Hands-on practice is essential to understand and fully grasp the concepts presented. The class is limited to 14 students.

What can I accomplish?

After finishing this intermediate-level course, you will create an online application using an enhanced knowledge of Web design.

Are there any prerequisites?

Students should have a fundamental knowledge of HTML and CSS. Students should take Web Basics 101 prior to enrolling in this course.
